# Bot de Figurinhas para WhatsApp

Bot em Node.js que transforma imagens e vídeos em figurinhas automaticamente no WhatsApp.

## Funcionalidades

- Envia uma imagem → vira figurinha automática
- Envia um vídeo curto → vira figurinha animada
- Comando `!sticker [url]` → transforma qualquer link em figurinha
- Comando `!ping` → verifica se o bot está online

## Como usar

```bash
npm install
node index.js
```

Escaneie o QR Code que aparece no terminal com o WhatsApp.

## Tecnologias

- Node.js
- whatsapp-web.js
- axios
- Puppeteer
